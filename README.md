# HOTAC-AI

## Overview
This is a fan hobby project that implements the AI maneuver system from **Heroes of the Arturi Cluster (HOTAC)** version 1.

## Purpose
HOTAC-AI recreates the automated opponent behavior mechanics from the classic space combat game, allowing players to engage in single-player or campaign-style missions against computer-controlled opponents using the original AI maneuver rules.

## Features
- Implementation of HOTAC v1 AI maneuver system
- Campaign-compatible gameplay
